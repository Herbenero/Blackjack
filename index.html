Blackjack darts

Tap scores. Apply doubles or trebles. Beat the House.

Every dart starts as a single. Use the number buttons from 1–20 plus Bull, or choose Miss. If 2x or 3x is selected, it multiplies the chosen number for that dart. The game lasts 8 rounds, throw order rotates, and the final thrower in each round is the House.

Single by default
2x / 3x as multipliers
Bust if round total exceeds 21
House: 3 pts if wins, 1 pt if ties

Players

Add at least 2 players. First throw rotates every round, and the last thrower is the House.
